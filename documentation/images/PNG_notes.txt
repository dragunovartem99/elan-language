Notes for PNGs referenced in HTML

Use 1920 x 1080 full screen of the IDE
at 72dpi resolution
Grab screen and crop as required
Store as PNG as LanguageAndIDE/documentation/images/filename.png
Determine Width and Height pixel counts: w1,h1
Calculate 70% of Width and Height pixel counts: w2,h2
Leave stored PNG unchanged
Use this HTML to size the PNG's display:
<img src="images/[filename.png]" width="[w2]" height="[h2]">
