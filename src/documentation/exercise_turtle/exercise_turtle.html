Learn procedural programming through turtle graphics

Step 1

Enter the following code:

# e50666c0705f49fd37a25a92e3d52ff4f798ba50580004e1e517e0ebb95639c5 Elan 0.10.0-Beta rwp
+main1 let tname? be new Turtle()expression?2 end main
new code procedure function test constant enum record class abstract interface #

and then run the program to see the output. Note that ...

  • The Display is 100 x 75 turtle units, and by default the turtle starts at the centre (x is 50, y 37.5).
  • The black line on the turtle shows its heading - by default the heading is 0 degrees.

After the first line within the main routine you can now add further instructions to the turtle by 'calling' procedures that are specific to a Turtle. If you add a call instruction and then type t. in the field, you will be given a list of those procedures.
